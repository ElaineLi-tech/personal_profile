【总体技能】
擅长打造三类完整线上产品：高稳定多人使用的后台服务、搭载 AI 自动处理功能的智能工具、微信小程序 / 手机网页轻量化应用，兼顾流畅使用、数据安全与趣味互动，快速落地可直接商用的成品项目。

【项目经验】
项目 1：AI 大模型垂直场景落地开发
技术栈
SpringBoot / LangChain4j / Milvus 向量库 / MongoDB / Redis / 通用大模型 API
能力概括
独立搭建垂直领域 RAG 智能对话客服系统，实现 7×24 小时自动化业务问答，承接订单、退款、配送等高频咨询，大幅降低人工客服转接压力。
AI 对话架构搭建：基于 LangChain4j 完成对话 Agent、提示词工程、多轮会话流程编排，对接并调优商用大模型；
RAG 知识库体系：业务文档分块、向量存储、语义检索优化，有效抑制 AI 幻觉，提升业务问答精准度；
业务工具联动：基于 Function Calling 封装后端业务接口，AI 可自动识别用户意图并执行订单查询、退款说明等实操业务；
会话与性能管控：MongoDB 持久化多轮对话上下文，Redis 缓存高频问答、接口限流兜底，保障高峰期服务稳定；
垂直 AI 工具落地：从零开发 AI 证件照生成机器人，接入大模型实现图片规格识别、证件照自动生成，日均稳定处理 300 + 用户请求，集成至移动端对外提供自助服务。

项目 2：AI 驱动跨端智能学习平台（Web SPA/PWA + 微信小程序）
技术栈
React + TypeScript + Zustand + TailwindCSS + Supabase BaaS + PostgreSQL + LLM API
能力概括
AI 辅助全栈开发，搭建 Web + 小程序双端一体轻量化学习产品，采用 BaaS 免运维架构快速完成 MVP 迭代，整合大模型生成内容、智能复习体系。
跨端架构设计：基于 MCP 协议打通 BaaS 前后端交互，统一 Web 与小程序业务逻辑层，一套业务逻辑适配双端，实现低成本快速上线；
AI 工程化开发流：搭建多角色开发 AI 智能体（前端 / 后端 / 测试 / UI），依托 PRD、数据库文档自动化拆分开发任务，大幅缩短迭代周期；
大模型接口封装与限流缓存：集成通用大模型 API 实现文本生成，设计多级语义缓存减少重复调用；增加单用户请求频率限制，拦截高频恶意请求，保障接口稳定；
工程自动化流程：标准化 Git 工作流，搭建 CI/CD 自动构建、测试、部署流水线，规范化版本迭代；
性能与数据安全治理：React Query 前端缓存、CDN 静态加速、PostgreSQL 复合索引 / 物化视图优化查询；基于 RLS 行级权限 + JWT 实现精细化数据隔离，保障多用户数据安全。

项目 3：轻量化家庭资产归档微信小程序
技术栈
微信原生小程序 + CloudBase 云开发（Serverless BaaS）
能力概括
独立从零完成物品管理小程序全流程设计与开发，面向个人收纳、藏品资产数字化管理，融合游戏化成长激励体系，兼顾离线可用、云端同步、数据导出完整闭环。
多维度结构化数据模型：设计十余个物品属性字段，支持多图上传、自定义分类 / 清单、多城市归档，提供卡片 / 表格双浏览视图，自动统计资产数量与总价值；
多租户数据隔离：多清单独立空间，分类、物品数据相互隔离，自定义分类配色，适配多场景收纳需求；
云端 & 离线双模式：依托云开发实现跨设备实时同步，本地持久化缓存支持离线操作，联网自动同步变更；支持 CSV 全量数据导出备份；
差异化游戏化成长系统：设计两条对立成长路线（收集 / 断舍离），分段段位解锁全屏动画特效、成就徽章墙；内置防刷限制机制，保证成长数值公平；
完整原生小程序能力落地：图片上传、本地存储、分享、自定义动画、下拉刷新、分页检索等微信原生 API 深度封装。
